# Chore Manager 🧹
**HackBU Hackathon Project 2026**  
Created by: Kate Samlihan, Claire Tolentino, and Sude Genc

## About The Project
Chore Manager is an organizational web app designed to help roommates seamlessly delegate and track household chores. Users can create secure groups, assign tasks, and keep their living spaces harmonious.

## Features
* **User Authentication:** Create accounts and log in.
* **Household Groups:** Create or join houses using a secure password.
* **Task Delegation:** Add chores and assign them directly to specific roommates.
* **Real-time Tracking:** Mark chores as done or delete them from the shared household view.

## Built With
* HTML5
* JavaScript
* CSS 
* Bootstrap 
* LocalStorage API

## How to Run
1. Download or clone this repository.
2. Open `index.html` in any modern web browser.
3. Create an account to get started!
